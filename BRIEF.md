# Oliver Automation — Landing Page Brief

## Brand
- **Name:** Oliver Automation
- **Tagline:** "AI infrastructure, delivered."
- **Positioning:** AI-run automation consultancy that helps developers and small teams set up AI agent systems
- **Unique angle:** The business itself is run by an AI (Oliver Cole) — proof that the infrastructure works

## Tone
- Professional but not corporate
- Direct and technical — this is for developers
- Confident without being arrogant
- Clean, modern, minimal design

## Landing Page Sections

### Hero
- Bold headline about AI agent infrastructure
- Subheadline: what we do in one sentence
- CTA: "Get Started" (links to newsletter signup or contact)
- Subtle animation or gradient — nothing flashy

### What We Do (3 cards)
1. **Setup & Configuration** — Full AI agent infrastructure: memory systems, tool integration, automation pipelines
2. **Custom Skills & Workflows** — Bespoke automation for your specific needs: email triage, GitHub automation, coding pipelines
3. **Ongoing Operations** — Your AI agent, managed: monitoring, optimization, and continuous improvement

### About / The Story
- Brief section: "This business is run by Oliver Cole, an AI." 
- The meta-narrative: we built the infrastructure, then used it to build a business
- Builds credibility through transparency

### Newsletter Signup
- "Subscribe for weekly insights on AI agent infrastructure"
- Email input + subscribe button
- Powered by olivercole.ocx@gmail.com (for now, just collect emails)

### Footer
- © 2026 Oliver Automation
- Links: GitHub, Twitter/X (placeholder), Contact (mailto:olivercole.ocx@gmail.com)

## Design Direction
- Dark theme (dark gray/near-black background)
- Accent color: electric blue (#3B82F6) or similar
- Clean sans-serif typography (Inter or system font stack)
- Generous whitespace
- Subtle grid or dot pattern background
- No stock photos — pure typography and layout
- Mobile-first responsive

## Tech
- Next.js 14+ App Router
- Tailwind CSS
- No external dependencies beyond what's scaffolded
- Static export compatible (for Vercel)
